# Product Overview

Geek Infiltration is a GitHub activity visualization application that displays GitHub timeline content (issues, pull requests, comments) in an elegant, accessible interface. Built as a modern React SPA with GitHub OAuth authentication.

## Core Capabilities

- **GitHub Timeline Display**: Aggregates and displays GitHub activity (issues, PRs, discussions) in a unified timeline view
- **GitHub OAuth Authentication**: Secure token-based authentication with automatic token persistence
- **Subscription Management**: User-configurable repository subscriptions for personalized content feeds
- **Landing Page Marketing**: Beautiful marketing landing page showcasing product features

## Target Use Cases

- Developers monitoring activity across multiple GitHub repositories
- Teams tracking project discussions and pull request activity
- Users wanting a consolidated view of GitHub interactions

## Value Proposition

A clean, performant interface for consuming GitHub activity without the noise of the full GitHub UI. Focuses on what matters: code discussions, issue tracking, and PR workflows.
